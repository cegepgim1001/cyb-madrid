Course Overview

Course Description

This 8-week intensive course is designed to equip cybersecurity students with the advanced, hands-on skills required to excel in modern security operations. The curriculum is a robust blend of three critical domains: security analysis, incident response, and proactive threat hunting. Students will learn to identify, analyze, and respond to threats using industry-standard tools and methodologies. The course culminates in a capstone exercise where students will apply their combined knowledge to a real-world breach scenario.

Learning Objectives

  • Analyze network traffic and system logs to identify malicious activity.
  • Triage and prioritize security alerts using a SIEM system.
  • Conduct live response and digital forensic analysis on endpoints.
  • Perform memory and network forensics to uncover evidence of compromise.
  • Develop and test hypotheses to proactively hunt for threats.
  • Utilize threat intelligence, such as the MITRE ATT&CK framework.
  • Document findings and create comprehensive incident response reports.
  • Build and manage a virtual lab environment for security testing.

Virtual Lab Setup

A foundational skill used throughout the course. Students must set up this VirtualBox environment before Week 1.

Software Requirements:

  • ▶Oracle VirtualBox (latest)
  • ▶Kali Linux VirtualBox Image
  • ▶Windows 10/11 Developer VM
  • ▶Security Onion VM

Key Steps:

  1. 1.Install VirtualBox & Extension Pack.
  2. 2.Configure a "Host-Only" virtual network.
  3. 3.Import/Install VMs with NAT & Host-Only adapters.
  4. 4.Take a "clean" snapshot of each configured VM.

Core Skills & Certification Alignment

The course is structured with a ponderation of 1:2:2, emphasizing Incident Response and Threat Hunting skills while building upon a strong Security Operations foundation. This approach prepares students for multiple roles and aligns with key OffSec certifications.

SOC-200 (OSDA) Focus

Skills in network/log analysis, SIEM operations, and alert triage.

IR-200 (OSIR) Focus

Mastering the IR lifecycle, live response, and deep-dive digital forensics on Windows/Linux.

TH-200 (OSTH) Focus

Developing a proactive mindset, creating hunt hypotheses, and using advanced techniques to find hidden adversaries.
